# **App Name**: GasyGo

## Core Features:

- Omni-Channel Order Entry: Multi-channel order placement via SMS, call, and WhatsApp. Orders are captured even when unstructured, by using a tool to interpret user intent. 
- Automated Order Confirmation: Automatic confirmation messages sent back to the customer via SMS/WhatsApp. ETA for deliveries estimated using a tool.
- Real-time Location Sharing: Customer-side location sharing with delivery team after order confirmation using geolocation API.
- Delivery Personnel Interface: Mobile app/web interface for delivery personnel showing customer location on a map with navigation.
- Inventory Management: Basic inventory management of gas cylinders and accessories with admin control to set pricing, initially using Firebase.
- Customer Database: Storage of customer names, phone numbers, addresses, and order history in the Firebase database.

## Style Guidelines:

- Primary color: Saturated orange (#FF8533) evokes the energy and heat associated with gas, while also feeling friendly and approachable, just like the TotalEnergies cylinders from the image.
- Background color: Light beige (#F2E7D9) provides a neutral backdrop that complements the vibrant primary color. This hue mirrors some of the muted concrete in the photo, to capture a bit of the spirit of Nairobi.
- Accent color: Deep green (#29A329) an analogous color drawn from the K-Gas cylinder, represents stability and eco-friendliness.
- Headline font: 'Poppins' (sans-serif) provides a clean, modern look for headlines.
- Body font: 'PT Sans' (sans-serif) provides a clean and readable typeface, suitable for user interfaces. Note: currently only Google Fonts are supported.
- Use simple, clear icons representing gas cylinders, accessories, delivery trucks, and location markers.
- Clean, intuitive layout with clear calls to action. Mobile-first responsive design.  Showcase prominent hero image.
